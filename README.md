# picEditor-
picEditor-369 is a client-side web application built using HTML, CSS, and JavaScript. It provides basic image editing tools such as resize, crop, border, quality adjustment, and image-to-JPEG/PDF conversion, along with utility features like a stopwatch and a real-time circular analog clock, without any backend or external libraries.
